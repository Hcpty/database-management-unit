# Readme
A note about TTMU (Typed Tablespace Management Unit).

### 分类的表空间管理单元

主存以页为单位，而数据库以表为单位。

Table Address数据结构：
- Resource Type
- Database Number
- Table Number

Record Address数据结构：
- Table Address
- Record Offset
