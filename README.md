# Readme
A note about Partitioned-Storage Database.

### 执行分区存储的数据库

把逻辑上属于同一个Collection的Resource分散存储到不同的Table中，而这些Table可能在不同的Database中，而这些Database又可能在不同的Database Node中，这种存储方式称为分区存储。
